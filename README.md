# Bankr-App-V3
Simple demo bank website with interactive features and buttons.

This website was created specifically to learn JavaScript and does not offer any additional functionality beyond the main page (you can't open an account).

However, you can interact with different buttons, see the sticky navigation bar when you scroll past the first section, and see the images slightly blurred in the second section before becoming clear (lazy loading).
Smooth scrolling is also implemented upon clicking on "Learn More" or the navigation menu.
There is a tabbed component in the operations section and a slider in the testimonial section as well.
